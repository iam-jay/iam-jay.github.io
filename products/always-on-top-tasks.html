Always-On-Top Tasks

A lightweight task board for Windows that stays pinned above every other window, so your to-dos are always in sight. Built for focus: a strict, forward-only workflow and honest, live overdue tracking that never lets a task quietly slip.

Features

Designed for focus

  • ✓Always on top. The window stays pinned above all others until you close it (with a toggle).
  • ✓Four clear stages. New, Active, Done, and Cancel — nothing more to think about.
  • ✓Forward-only flow. Tasks move New → Active → Done/Cancel and never go backward.
  • ✓Extend-only timelines. Once set, a deadline can only be extended — shown with an "Extended" badge.
  • ✓Live overdue tracking. Past-due tasks show a red, ticking OVERDUE counter.
  • ✓Hour-based scheduling. Activating a task sets the start to now and asks for a duration in hours.
  • ✓Saves automatically. Everything is stored locally and restored on restart.
